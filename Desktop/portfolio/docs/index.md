# Marco's Portfolio

Ce projet combine génération d’images par IA, impression 3D et robotique afin d’automatiser la création de canards personnalisés.

L’objectif est de générer des designs de canards à l’aide de modèles d’IA, de les transformer en textures appliquées sur un modèle 3D, puis d’utiliser un bras robotisé pour peindre automatiquement les canards imprimés en 3D.

Le projet explore différentes méthodes de génération, ainsi que la mise en place d’un pipeline permettant de générer, évaluer et appliquer ces designs de manière automatisée.
